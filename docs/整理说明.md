# UAV Platform 整理说明

## 目录分类

- `frontend/`: 无人机平台前端项目。
- `backend/`: 后端服务、邮件服务、推流服务。
- `algorithems/`: 航线生成、识别分割、YOLO/SAM/TransFuseNet 等算法项目。
- `docs/`: 整理记录和原项目树文件。

## 已优化内容

- 清理了可再生成的依赖和构建产物：`node_modules/`、`dist/`、`target/`、`.m2/`。
- 清理了 IDE 配置、运行日志和 Python 缓存：`.idea/`、`.vscode/`、`logs/`、`log/`、`__pycache__/`。
- 对重复的大模型权重和测试视频使用 NTFS 硬链接保留原路径，减少物理磁盘占用，同时不改变代码引用路径。

## 重新生成提示

- 前端依赖：在 `frontend/Cloud-API-Demo-Web-main` 下运行 `npm install`。
- 前端构建产物：在前端目录运行项目原有构建命令生成 `dist/`。
- Maven 项目构建产物：在对应 Java 项目目录运行 `mvn package` 或使用项目自带 `mvnw`。
